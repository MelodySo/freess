<details>
    <summary>使用说明和建议</summary>

```
* 倍率的意思：假如倍率是10，实际使用1G的流量，会消耗10G的流量余额。倍率0就是不消耗余额
* 限速：节点可以限速，账户也可以限速。即使机场不主动限速，速度也会受很多其它因素影响（木桶原理）
* 延迟和速度是两个概念。延迟低且速度快当然是最好的。但若延迟一般，速度仍可能很快，反之也一样成立
```
